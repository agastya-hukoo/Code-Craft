Welcome to CodeCraft

CodeCraft is a streamlined Windows Forms application designed for real-time HTML editing and previewing. It integrates a text editor, browser preview, and custom settings, enhancing your coding experience with a sleek, dark interface.

Quick Start

Edit & Preview: Write HTML on the left and see it live on the right. Use the toolbar for file operations and settings. Customize themes and music from the settings menu.

Dependencies

  • .NET Framework: Latest version.
  • CefSharp: For browser preview.
  • FastColoredTextBox: For enhanced text editing.
